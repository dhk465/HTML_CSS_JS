White-space

If not allowing line changes, it can print a line without adding any lines. When the text is longer than the web browser window, it creates a scroll bar.

for (int i = 0; i < 3; i++) { System.out.println(i); }

It is defined in "text-overflow". If the text is longer than the web browser window, it changes the rest of the text into an ellipsis. It requires both overflow(hidden) and white-space(no-wrap) already defined in CSS.
